# garlandTilt
ESP32 based tilt sensor with ring led display
#### Features
* uses MPU-6050 as accelerometer
* neopixel ring with 60 RGB LED
* auto calibration
#### I2C bus
* SDA GPIO 17
* SCL GPIO 16
* address 0x68
#### led ring serial in
* GPIO 15
